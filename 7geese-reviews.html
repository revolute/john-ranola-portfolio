Improving the performance review: a collection of UI designs

7Geese

A collection of various UI designs for different projects during my time at 7Geese. I was the lead designer for the Performance Reviews product, reflected in the work you see below.

Project

Reviews dashboard redesign

What is it

An evolution of the dashboard for the Reviews feature. This was driven by the introduction of several new features, and to address long-standing limitations with the original design.

Design highlights
  1. Focused the page around a to-do list paradigm. My rationale was that the user’s primary need for this page is to know the most important task to do next.
  2. “Your latest review”: Consolidated the active and completed reviews sections for a more appropriate use of space.
  3. Status tags for the latest review to communicate next action at a glance.
  4. Pattern reuse: avatar bubbles to show the user all reviews needing their attention for a particular stage.
Project

Peer feedback in Reviews

What is it

A major project integrating 7Geese’s Feedback and Reviews features. This allowed a performance review to incorporate feedback from peers, instead of being just between an employee and their manager.

This is seen as a best practice in the performance management community as it promotes a fairer, less biased performance review.

Design highlights
  1. A redesign of the progress tracker for a review which better illustrates that a review can be in several steps in parallel. Previously the tracker used a linear graph.
  2. Tab navigation trimmed to support variable tabs within a review. This is because peer feedback is an optional feature of a review.
